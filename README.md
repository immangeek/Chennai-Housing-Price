## Run Locally

Clone the project

```bash
  git clone https://github.com/immangeek/Chennai-Housing-Price.git
```

Go to the project directory

```bash
  cd chennai-housing-price
```

Install dependencies

```bash
  pip install streamlit
```

Start the server

```bash
  streamlit run app.py
```
